# Porsline Telegram Reporter

این سرویس پاسخ‌های دو فرم پرسلاین را می‌خواند، فقط ردیف‌های جدید را در قالب اکسل قرار می‌دهد و هر ۱۴ روز فایل را به یک گروه خصوصی تلگرام می‌فرستد.

## قواعد خروجی

- ترتیب ردیف‌ها: تزریقات، سپس تکنسین داروخانه
- ردیف‌های تکنسین: زرد
- ستون‌ها: نام فارسی، کد ملی فارسی، کنترل صحت کد ملی، نام انگلیسی، کد ملی انگلیسی
- نام فایل: `تیر-تکنسین x-y.xlsx`
- `x`: تعداد کل پاسخ‌های فرم تزریقات
- `y`: تعداد کل پاسخ‌های فرم تکنسین
- جلوگیری از تکرار بر پایه شناسه پاسخ‌دهنده؛ اگر شناسه در پاسخ API نبود، هش پایدار کل پاسخ استفاده می‌شود.

## متغیرهای محیطی

مقادیر `.env.example` را در بخش Variables سرویس میزبانی وارد کنید. فایل `.env` را commit نکنید.

## اجرا

```bash
pip install -r requirements.txt
gunicorn --workers 1 --threads 4 --timeout 180 app:app
```

سلامت سرویس در `/health` و اجرای دستی حفاظت‌شده در `POST /run-now` با هدر `X-App-Secret` قابل بررسی است.
